# 📊 Mobile Sales Analytics Dashboard | Power BI

An interactive **Mobile Sales Analytics Dashboard** built using **Microsoft Power BI** to analyze sales performance across brands, mobile models, cities, payment methods, and time periods. This project demonstrates the use of Power BI for transforming raw data into actionable business insights through interactive visualizations.

## 🚀 Project Overview

This dashboard was built to practice and apply core Business Intelligence concepts using Microsoft Power BI.

It enables users to explore sales performance through interactive filters and visualizations, making it easier to identify trends, compare products, and support data-driven decision-making.

---

## 📌 Key Features

- 📈 KPI Cards for Total Sales, Quantity, Transactions, and Average Sales
- 📅 Monthly Sales Trend Analysis
- 🗺️ Geographic Sales Analysis
- 📱 Mobile Model Performance Analysis
- 💳 Payment Method Analysis
- ⭐ Customer Ratings Analysis
- 🎛️ Interactive Slicers for dynamic filtering
- 📊 Interactive Business Dashboard

## 📈 Business Questions Answered

This dashboard helps answer questions such as:

- Which mobile brand generated the highest sales?
- Which mobile models performed the best?
- How do sales vary across different cities?
- What are the monthly sales trends?
- Which payment methods are most frequently used?
- How are customer ratings distributed?
